软件推荐
-------

### 办公

- Keynote - 这才是 PPT 神器
- Office 2011(word, ppt, excel) - 不用在虚拟机里面折腾了
- PreziDesktop - Prezi，ppt神器二号
- Adobe Acrobat - 编辑pdf
- TimeLine 3D - 基于时间轴的展示

### 阅读写作

- Preview - 浏览pdf
- iChm - Chm 阅读
- Kindle - Mobi 阅读
- Murasaki - ePub 阅读
- OmmWriter - 小黑屋写作
- Mou - Markdown 写作

### 社交网络

- QQ - Mac 下依旧是 CPU 占用前十的渣渣
- 阿里旺旺 - 淘宝
- Skype - 团队会议用
- Messages - Google Talk and iMessage
- Tweetbot - twitter 最好的客户端，没有之一
- Limechat - irc

### 影音媒体

- iTunes - OS X 自带的
- DynamicLyrics - 配合 iTunes 显示歌词
- MplayerX/Movist  - 视频播放器
- SmartConverter - 视频格式转换

### 图像相关

#### 图像编辑

- Adobe Photoshop CS6 - 万能
- Pixelmator - PS精简版
- Sketch - UI
- OmniGraffle - 作图软件，各种流程图类图

#### 配色工具

- ColorSchemer Studio 2 - 配色方案工具
- Spectrum - 配色方案工具
- Pochade - 取色工具


### 知识管理

#### 日常

- Evernote - 笔记管理
- Dropbox - 文件同步
- Aperture/iPhoto - 照片管理，有单反的就用 Aperture，没的话 iPhoto 足够了
- iDocument - 电子文档管理工具。Calibre 用户体验太差不推荐，虽然他是开源跨平台的软件
- Airmail/Mailplane - Gmail 桌面 app

#### 设计湿 or 程序猿 or Blogger

- Sparkbox - 设计素材管理
- Dash/Snippets/Codebox - 前者免费，后者收费，代码片段管理 app
- Dash - 文档管理
- MarsEdit - WordPress 写作 app

#### 个人知识管理

- MindNode Pro - 思维导图
- OmniFocus/Things - GTD
- Pocket - Pocket 的桌面 App
- Reeder - 看 Google Reader
- Delish - 个人书签服务 Pinboard.in 的桌面 App
- 1Password - 密码管理+自动填写工具

### 开发相关

#### Editor/IDE

- AppCode - Obj-C IDE
- IntelliJ IDEA - Java IDE
- PyCharm - Python IDE
- RubyMine - Ruby IDE
- iTerm2 - 最好的终端
- HomeBrew/Cakebrew - 最好的包管理工具，Cakebrew 是其 GUI 版本
- MacVim - Mac下的vim（HomeBrew安装），安装后继续用他的 cli 版，因为编译的时候有添加对 python/ruby/perl 的支持
- Sublime Text 3 - vim killer
- Kaleidoscope - 漂亮易用的 diff 工具

#### 版本管理

推荐用 git 命令行版，如果不是开发者用 gui 的也不错。

- SourceTree - Git/Hg 版本管理工具，而且还免费，Atlassian 是家良心公司。

#### 网络开发

- Wireshark - 万能抓包工具
- HTTPScoop - HTTP 分析
- Chales - HTTP 分析工具，java 写的
- Rubbernet - 网速监控
- Speed Limit - 模拟低速网络状况

#### Web开发

- Espresso - web 开发编辑器
- LiveReload/Codekit - coffee/haml/sass 等实时编译 与 html 实时预览
- Smaller - html/js/css minify
- Postman - REST API 测试
- VirtualHostX - Apache gui 管理工具

#### 数据库

- MesaSQLite - SQLite
- MongoHub - MongoDB
- Sequel Pro - MySQL
- Navicat Data Modeler/SQLEditor - 数据库建模

#### 其他

- Patterns - 正则表达式工具
- Xmplify - XML 工具
- CodeRunner - 支持各种语言的代码片段编写工具


### 实用工具


#### 功能加强

- Alfred - Spotlight 加强
- BetterDictionary - 系统自带词典插件，增加词库功能
- Fantastical - 日历增强
- AppCleaner - 更完全的卸载软件
- AppFresh - 检测软件更新
- BettertouchTool - 触控板加强
- Moom - 窗口管理
- Paragon NTFS - NTFS 写支持
- ForkLift - FTP + 文件管理
- BetterZip - 支持rar
- Thunder - 迅雷
- iStat Menus - 在系统状态栏显示 cpu 和网速等信息
- CheetSheet - 显示当前软件的快捷键
- Parallels Desktop - 虚拟机，性能不错
- ClipMenu - 剪切板管理器

#### 系统清理

- Gemini - 找到系统里面的重复文件
- CleanMyMac - 垃圾清理
- DaisyDisk - 饼状图显示磁盘空间占用

#### 性能测试工具

- Blackmagic Disk Speed Test - 测试磁盘性能
- Geekbench - CPU 内存等综合测试的工具

#### 其他

- DMGMaker - DMG 包制作
- Caffeine - 不要休眠
- CandyBar - 修改软件图标美化
- SceenFlow - 录视频
- KeyCastr - 显示你的当前按键，录视频有用
- Droplr - 分享文件和图片的工具，Gtalk 发人家图片可以利用它
- GoAgentX - 这个你懂的
- Weather HD - 看天气的软件
- Onyx - 系统维护及优化程序，新手绕道

### 逆向工程

- idaq - IDA 在 OS X 下的 native app
- otx - otool 的 gui app
- 0XED - hex 编辑器
- Hex Friend - hex 编辑器
- Hopper Disassembler - 反汇编工具
